为Nginx配置SSL证书

发表于 2015-10-18   |   分类于 技术   |  

前言

现在各大站点都开始使用https来保证网站的安全，我也不例外。
这次我主要讲下如何配置ssl证书并设置自动跳转。

证书

要配置SSL前当然需要一张证书，我是在淘宝买的6元一年，很便宜的说。

购买地址：https://item.taobao.com/item.htm?spm=a1z09.2.0.0.JvzwVQ&id=15823207240&_u=51qc251f8395

开启SSL支持

我以Nginx的默认文件为例

nano /etc/nginx/sites-available/default
在server{}之间添加以下内容

ssl on;
ssl_certificate .crt文件路径;
ssl_certificate_key .key文件路径;

自动跳转到http

将以下内容粘贴到原本的server{}之前

server {
server_name 你的网址;
rewrite ^(.*)$ https://$host$1 permanent;
}

重启nginx service nginx restart

结语

配置SSL是比较简单的，按照教程来就可以实现，有任何不懂的地方可以给我留言。
顺便说下Nginx需要自己把中继证书中的内容粘贴到.crt文件，否则在手机上访问会出现证书问题
