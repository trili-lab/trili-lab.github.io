Ubuntu Mate 用fcitx框架安装中文谷歌输入法

发表于 2016-04-16   |  

刚装完Ubuntu Mate时是默认的输入法是Fcitx，但是并没有提供中文输入。
1、先卸载默认安装的ibus(如有安装):
sudo apt-get remove ibus
2、如果之前安装过Fcitx，先将其卸载：
sudo apt-get remove fcitx*

找回失去的add-apt-repository

发表于 2016-04-16   |  

想给VPS添加个PPA，然而大多VPS运营商提供的系统都没有add-apt-repository。。
都是精简版系统惹的祸= =
apt-get install python-software-properties
apt-get install software-properties-common
然后就能用add-apt-repository来方便快捷的添加PPA了。
