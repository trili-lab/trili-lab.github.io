找回失去的add-apt-repository

发表于 2015-10-18   |   分类于 备忘   |  

想给VPS添加个PPA，然而大多VPS运营商提供的系统都没有add-apt-repository。。
都是精简版系统惹的祸= =
apt-get install python-software-properties
apt-get install software-properties-common
然后就能用add-apt-repository来方便快捷的添加PPA了。
